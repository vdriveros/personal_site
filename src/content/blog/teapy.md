A Laravel-based medical appointment management system designed for healthcare providers to manage patient appointments and medical records efficiently.

**Description:**
Healthcare-focused web application for managing medical appointments, patient information, and appointment scheduling. Built with Laravel framework with a user-friendly interface for both administrators and patients.

**Technologies:** PHP, Laravel, Livewire, Blade Templates, MySQL, JavaScript

**Key Features:**
- Patient appointment scheduling
- Medical record management
- Appointment calendar and status tracking
- Doctor/staff management
- Secure patient data handling
- Responsive interface for desktop and mobile access
