A custom WordPress theme developed for an international university with a modern, responsive design using Tailwind CSS.

**Description:**
Premium WordPress theme designed specifically for educational institutions with comprehensive curriculum management, program listings, and institutional information display. Built with modern development practices using Tailwind CSS for styling and ACF (Advanced Custom Fields) for flexible content management.

**Technologies:** PHP, WordPress, Tailwind CSS, ACF (Advanced Custom Fields), JavaScript, HTML/CSS

**Key Features:**
- Custom program/career listings with detailed templates
- Advanced Custom Fields (ACF) integration for flexible content
- Responsive design with Tailwind CSS
- University news and announcements section
- Program and course information architecture
- Optimized for search engines
- Modern, professional university branding

**Theme Customizations:**
- Archive pages for programs and courses
- Custom single page layouts
- Front-page builder integration
- Asset management system
